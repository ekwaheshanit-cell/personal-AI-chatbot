# 🤖 Personal AI Chatbot Website

A beginner-friendly AI chatbot web application built using **Claude Code**, following a structured **PRD (Product Requirement Document)** approach.
This project demonstrates how to create, test, and iterate a simple web application using GitHub — without prior coding experience.

---

## 📌 Project Overview

This project is a simple chatbot website where users can type messages and receive AI-generated responses through a clean and minimal interface.

It is designed as a **learning project** to understand:

* How AI-assisted development works
* How to use Claude Code for building applications
* How to manage projects using GitHub
* How to follow a PRD-driven workflow

---

## 🎯 Objectives

* Build a functional chatbot UI
* Practice PRD-based development
* Learn GitHub workflow (commit, push, update)
* Explore no-code / AI-assisted development

---
