# Product Requirement Document (PRD)

## 1. Product Name
Personal AI Chatbot Website

## 2. Goal
Create a simple website where users can chat with an AI assistant.

## 3. Target User
- Beginners
- Personal use

## 4. Features

### Core Features
- Chat input box
- Send button
- AI response display
- Clean UI

### Optional Features
- Dark mode
- Chat history (basic)

## 5. User Flow
1. User opens website
2. User types message
3. Clicks send
4. AI responds

## 6. Design Style
- Minimal
- Modern
- Centered chat box

## 7. Tech (Handled by Claude Code)
- HTML
- CSS
- JavaScript

## 8. Constraints
- No complex backend
- Beginner-friendly
